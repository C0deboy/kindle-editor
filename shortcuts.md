# kindle-editor

## Useful shortcuts defined in raspberry pi.

Location: `/home/pi/.config/openbox/lxde-pi-rc.xml`

> `CTRL` + `SHIFT` + `ALT` + `K`

Starts this application

> `WINDOWS` + `END`

Shutdown raspberry pi

> `WINDOWS` + `BACKSPACE`

Restart raspberry pi

## In app shortcuts

> `CTRL` + `S`

Save file
